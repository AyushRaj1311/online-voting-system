ONLINE VOTING SYSTEM

An Online Voting System is a web-based application that allows users to vote electronically in a secure and efficient way. This system helps in conducting elections digitally by reducing paperwork, saving time, and increasing transparency.

FEATURES
- User Registration & Login
- Admin Dashboard
- Voter Authentication
- Secure Voting Process
- Real-time Vote Counting
- Result Declaration

TECHNOLOGIES USED
Frontend: HTML, CSS, JavaScript
Backend: Java / Spring Boot
Database: MySQL

HOW TO RUN
1. Clone the project from GitHub
2. Open in IntelliJ IDEA
3. Configure MySQL database
4. Run the application
5. Open browser and go to http://localhost:8080
